# RAINING-TACOS
Everyone in town was enjoying the tacos until tacos started coming down like hail! can you survive the taco haul before its too late?
-----------------------------------------------------
CONTROLS:
Press the right or left arrow key to move right!
You an also use the a and d keys to move!
Click to use things!
-----------------------------------------------------
HOW TO PLAY:
Avoid the falling tacos in the sky by moving around! If you get hit, its game over for you! Try to hit high scores and have fun! (PS: srry about the glitches, this is a beta game after all!)
